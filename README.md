# Alaca Info
Pasif bilgi toplama aracı.

## Özellikler
- Whois<br>
- DNS Lookup<br>
- Geoip<br>
- Subnet Lookup<br>
- Host Finder<br>
- Page Links<br>
- Robots.txt<br>

## Kullanılan Kütüphaneler
- os<br>
- rquests<br>
- optparse<br>
- socket<br>
- whois<br>

## Kurulum
`git clone https://github.com/wol4n/alaca-info`

`cd alaca-info`

`pip install python-whois`

`python alaca-info.py`
